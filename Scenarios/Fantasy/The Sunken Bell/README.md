# The Sunken Bell (Scenario Pack)

A standalone EMBER Fantasy scenario pack — not part of a series.

This folder has two files:

- **`The Sunken Bell - Player.md`** — read this one. Safe, spoiler-free. Paste it to your AI along with `EMBER — Core Engine.md` (Fantasy is baked in, so no Genre Pack needed) to start the scenario.
- **`The Sunken Bell - GM (Spoilers).md`** — **don't read this one.** It's written for the AI Game Master and contains the villain, the true backstory, and the resolution. Reading it yourself spoils your own game, since EMBER is solo play and there's no second human to keep it hidden from you.

**How to run it:** paste `EMBER — Core Engine.md`, `The Sunken Bell - Player.md`, and `The Sunken Bell - GM (Spoilers).md` to your AI, and say: *"Run Session Zero using these — I'm playing The Sunken Bell scenario."* The AI will use the Player doc to frame the hook for you and the GM doc to run everything behind the scenes.

If you're on a setup that supports hidden project knowledge (see `PLAYING-ON-DESKTOP.md` / `PLAYING-ON-MOBILE.md`), attach the GM doc there instead of pasting it into the visible chat — then you genuinely never have to see it.
