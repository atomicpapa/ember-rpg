# Characters

One file per persistent character, created and updated by the AI during play.

Each character is stamped with a **genre** at creation and can only enter scenarios of that genre (any *lens* of that genre is fine). Characters carry their XP, learned moves, permanent tags, and earned gear forward between scenarios. See the persistence rules in `EMBER — Core Engine.md`.

Copy `../Character Sheet Template.md` to start a new one.
